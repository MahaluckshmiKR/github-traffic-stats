# github-traffic-stats

## Python CLI

Python CLI tool to get stats on the command line.

### Dependencies

- [PyGithub/PyGithub](https://github.com/PyGithub/PyGithub) 

### Run

Run on the command line.

```
$ python github-stats.py 'nchah' 'github-traffic-stats'
Password:*

```
